# Quantum Computing Simulation using Qiskit

## Project Description
This project demonstrates quantum computing concepts using Qiskit.

## Features
- Superposition
- Entanglement
- Quantum Coin Flip
- Deutsch-Jozsa Algorithm

## Tools Used
- Python
- Qiskit
- Jupyter Notebook / Spyder
- IBM Quantum Experience

## How to Run
1. Install Qiskit:
   pip install qiskit qiskit-aer matplotlib

2. Run the file:
   python quantum_project.py

## Results
- Probabilistic outputs
- Quantum faster than classical in some cases
